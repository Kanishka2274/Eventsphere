# EventSphere

EventSphere is a multi-event management platform that allows users to discover, manage, and register for multiple events from a single dashboard.

## Problem
Event organizers often struggle to manage multiple events, track participants, and share event information efficiently. Participants also find it difficult to discover and register for events across different platforms.

## Solution
EventSphere provides a centralized platform where organizers can create and manage events while participants can easily browse and register for events.

## Features
- View multiple upcoming events
- Event cards with details (date, location, participants)
- Event registration system
- Real-time participant tracking
- Simple and user-friendly dashboard
- Event feedback with sentiment analysis
